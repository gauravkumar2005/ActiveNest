# 🏋️ ActiveNest

An interactive fitness website built with **HTML, CSS, and JavaScript** that helps users achieve their health and wellness goals through personalized workout programs, fitness classes, and expert guidance.

🔗 **Demo:** https://active-nest.vercel.app/

## ✨ Features

* 🏋️ Explore fitness programs and workout plans

## ⚙️ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/gauravkumar2005/ActiveNest.git
```

### 2. Navigate to the project directory

```bash
cd ActiveNest
```

### 3. Open the project

Since this is a static website, simply open the `index.html` file in your browser.

Alternatively, you can use the VS Code Live Server extension for a better development experience.
